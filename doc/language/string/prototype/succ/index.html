                                                                                      instance method String#succ 

                                                                                      String#succ() → String

                                                                                      Used internally by ObjectRange.

                                                                                      Converts the last character of the string to the following character in the Unicode alphabet.

                                                                                      Examples
                                                                                      'a'.succ();
// -> 'b'
 'aaaa'.succ();
// -> 'aaab'
